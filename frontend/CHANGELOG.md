# Changelog

## [0.1.0] — 2026-05-12

### Added
- Initial release of Cortex Interface
- Chat UI with streaming (SSE) and non-streaming toggle
- Session sidebar with history loaded from backend
- Dark / light theme with system preference detection and localStorage persistence
- Glassmorphism floating prompt bar with auto-resize textarea
- Lavender design system aligned with DESIGN.md tokens
- Docker support: multi-stage `node:20-alpine` → `nginx:alpine`, served on port 8080
